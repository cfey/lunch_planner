# Lunch Planer Web Frontend

## Installation

```bash
npm install
```

## Development

```bash
npm run start
```
## Production

```bash
npm run build
```
## Linting

```bash
npm run lint
```

## Testing

```bash
jest __tests__ --watch
```

# WebStorm Configuration
## Plugins

* install [ESLint](https://plugins.jetbrains.com/plugin/7494-eslint) and enable it in the settings
